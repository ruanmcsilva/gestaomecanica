O projeto adota a estrutura de monorepositório, onde o código-fonte do Backend (Django) e do Frontend (React Native/Web) são orquestrados por um arquivo docker-compose.yml na raiz do diretório. Essa padronização visa simplificar o processo de instalação e garantir que todas as dependências de hardware e software detalhadas no requisito [RNF03] sejam atendidas automaticamente via containers.
